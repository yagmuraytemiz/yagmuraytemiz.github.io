Project Overview

Summary

In this project, an experiment is designed to calculate the corresponding lengths of tubes needed to produce certain pitches. While learning the topics that I needed to know to design the experiment, I encountered many points that could cause misinterpretations, and also saw that many resources didn’t connect physics and music. So, I started to write summaries for topics that focus only on the misinterpretations and combining physics and music using fundamental keywords and creating a path for further research.

Goal

Although physics and music are interconnected fields, most of the resources don't connect these topics. The goal of this project is to explain the links between these two fields and create a resource that focuses on these links and misinterpretations and creates an outline that can be used for deeper research.

Misinterperations

  • Doubling the frequency will double the speed of sound
  • When a note is played from an instrument, only the first harmonic is produced
  • Equal temperament scale is more accurate
  • Wavelength and period refer to the same property of a wave
